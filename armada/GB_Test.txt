Настройка и подготовка тестовой среды для reactJs.

1)  Скачиваем NodeJS модуль по ссылке под свою ОС https://nodejs.org/ru/

2)  Чекаем наличие ноды через терминал командой: node -v (должна вернуться версия v14.17.5)

3)  Переходи в дирректорию armada

4)  В терминале пиши команду npm install --save-dev sass-loader@7.1.0

5)  Затем: npm install react-bootstrap@next bootstrap@5.1.0

6)  Затем: npm install react-router-dom

7)  Проверь package.json должно быть так: 
    "name": "armada",
    "version": "0.1.0",
    "private": true,
    "dependencies": {
        "@testing-library/jest-dom": "^5.14.1",
        "@testing-library/react": "^11.2.7",
        "@testing-library/user-event": "^12.8.3",
        "google-map-react": "^2.1.10",
        "google-maps-react": "^2.0.6",
        "iframe-react": "^1.0.1",
        "react": "^17.0.2",
        "react-dom": "^17.0.2",
        "react-scripts": "4.0.3",
        "web-vitals": "^1.1.2"
  },
  Это карта зависимостей, если чего то нет смело вписывай в гугл название и приписывай npm install интернет тебе поможет, 
  при наличии всех зависимостей сборка будет коректно отображать все компонеты... 

8)  Затем после всех танцев пиши: npm start   и жди когда запустится браузер.

Если не будет работать, то пиши в GB телегу Кузнецову Леониду.
